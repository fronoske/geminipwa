# 手動スモークテスト

## 実施前

- 対象コミット、ブラウザ、OS、画面幅を記録する。
- 既存データを設定画面からエクスポートする。
- 通常プロファイルでは既存データからの起動を、別プロファイルでは初回起動を確認する。
- API送信テストは、原則として無料モデルまたはDummy AIを使用する。

## 起動とデータ互換性

- [ ] 初回起動でチャット画面が表示され、コンソールに未処理エラーがない。
- [ ] 既存プロファイルで設定と会話履歴が維持される。
- [ ] エクスポートしたデータをインポートできる。
- [ ] 再読み込み後も設定と会話履歴が維持される。
- [ ] IndexedDB名 `GeminiPWA_DB` とスキーマバージョン8が意図せず変わっていない。

## チャット

- [ ] 新規チャットを開始できる。
- [ ] ユーザーメッセージとDummy AI応答を保存できる。
- [ ] 過去のチャットを読み込み、編集、再送、複製、削除できる。
- [ ] ストリーミング表示と中断操作が機能する。
- [ ] テキスト・画像添付の追加と削除が機能する。
- [ ] Markdown、コードブロック、Mermaid表示に重大な崩れがない。

## 画面と設定

- [ ] チャット、履歴、設定の各画面をボタンとスワイプで移動できる。
- [ ] 設定変更が即時反映または保存操作後に反映される。
- [ ] テーマ、文字サイズ、背景画像、ヘッダー・フッター設定が再読み込み後も維持される。
- [ ] 代表幅360px、768px、1280pxで横方向の意図しないはみ出しがない。
- [ ] キーボード操作時にフォーカス位置が判別できる。

## 補助機能

- [ ] メモ、クリップボードスタック、校正機能が開閉・保存できる。
- [ ] APIプロバイダー切替とAPIキー切替が機能する。
- [ ] 設定と履歴のインポート・エクスポートが機能する。

## PWA

- [ ] Service Workerが登録され、`index.html`、`manifest.json`、`marked.js`、アイコンがキャッシュされる。
- [ ] オンラインで一度起動した後、オフライン再読み込みで画面が表示される。
- [ ] キャッシュ消去操作後に最新版へ更新される。
- [ ] インストール済みPWAから起動でき、表示モードとアイコンが正しい。

## 入力プリセット

- [ ] 設定画面でプリセットを追加・編集・削除でき、再読み込み後も維持される。
- [ ] プリセット本文の改行が入力欄へそのまま挿入される。
- [ ] `{|}`が本文から除去され、その位置へカーソルが移動する。
- [ ] `{|}`がない場合は挿入内容の末尾へカーソルが移動する。
- [ ] 自動送信を有効にしたプリセットが挿入後に送信される。

## 入力・出力移動ボタン

- [ ] 上矢印で前、下矢印で次のメッセージ末尾へ移動する。
- [ ] 上矢印は先頭メッセージ末尾の次に画面最上部へ移動し、最上部では無効になる。
- [ ] 入力欄が複数行になっても、ボタンが入力欄の上に表示される。
- [ ] 「常に表示」「非表示」「スクロール時のみ表示」が再読み込み後も維持される。
- [ ] 「スクロール時のみ表示」ではスクロール開始時に現れ、停止後に隠れる。

## ヘッダーメニュー改修後に追加する確認

- [ ] 「新」「削」「コ」がヘッダーに表示されない。
- [ ] ハンバーガーメニューから「新規チャット」「全削除」「全コピー」を実行できる。
- [ ] メニューは外側タップ、Escape、項目選択で閉じる。
- [ ] ハンバーガーボタンの表示設定が再読み込み後も維持される。
- [ ] 「全削除」で先頭項目だけが残る。
